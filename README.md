## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pritam-purkait/NewsFeed-App.git
   ```
2. Install dependencies:
   ```bash
   cd NewsFeed-App
   npm install
   ```


## Running the App

```bash
npm start
```
The app will run on `http://localhost:3000`

## Running Tests

```bash
npm test
```
